# Raw Essentials public surfaces

- Catalogue: `GET https://rawessentials.co.nz/products?petType=dog|cat&search=<query>&page=N`
- Detail: public same-site `/product/...` pages
- Stores: `GET https://rawessentials.co.nz/stores`

The site exposes public server-rendered HTML, not a documented retail API. Extraction is intentionally shallow and bounded. Page HTML and markup can change; include returned source URLs and timestamps, and do not use catalogue content as veterinary or nutritional advice.
