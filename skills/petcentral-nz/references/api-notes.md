# Pet Central public surfaces

- Catalogue: `GET https://petcentral.co.nz/products.json?limit=250&page=N`
- Detail: `GET https://petcentral.co.nz/products/<handle>.js`
- Collections: `GET https://petcentral.co.nz/collections.json?limit=250`

These are Shopify public JSON surfaces observed July 2026. They are unauthenticated, read-only discovery surfaces, not a stable contractual API. The CLI limits page and result sizes and sends a descriptive user agent. Do not infer medical suitability or complete any commerce workflow.
