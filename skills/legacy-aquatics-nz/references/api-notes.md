# Legacy Aquatics public surfaces

- Search: `GET https://legacyaquatics.co.nz/?s=<query>&post_type=product&paged=N`
- Category: `GET https://legacyaquatics.co.nz/product-category/<slug>/[page/N/]`
- Detail: public same-site `/product/...` pages

These are public WooCommerce/WordPress HTML surfaces, not a documented stable API. The connector only performs bounded GET requests and shallow extraction. Product text and prices are snapshots, not aquarium/reptile fitment, welfare, safety, or husbandry advice.
